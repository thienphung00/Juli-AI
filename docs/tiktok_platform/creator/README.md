## Quick-reference: Creator account type gates

| Account Type | Min. Followers | Earn from Video/LIVE | Showcase visible | Free samples |
|-------------|---------------|---------------------|-----------------|-------------|
| Link-Share Only | 0 | No | Hidden | No |
| Affiliate Creator | ≥ 1,000 | Yes | Yes (public) | Yes |
| Shop Official Account | 0 | No (own shop only) | Shop page | — |
| Shop Marketing Account | 0 | Own shop + affiliate-eligible once criteria met | Showcase | — |

Source: [Creator E-commerce Permissions Eligibility](https://seller-vn.tiktok.com/university/essay?knowledge_id=104015979497218&lang=en)
